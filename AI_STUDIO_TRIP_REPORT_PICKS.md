# TRIP REPORT SITE — DESIGN PICKER (for Google AI Studio)

Companion to `MOOD_BREAKDOWN.md`. This is the opinionated shortlist: which of the 164 designs in this repo belong on a website about tripping, and exactly what job each one should do. Hand this whole file to AI Studio as project context, then point at sections when prompting.

---

## THE PITCH

A trip-report site needs more than pretty shaders — it needs a **perceptual system**: a hero that replaces reality, dividers that mark narrative state-changes, ambient layers that breathe behind long text without fighting it, interactive toys for the reader's own pattern-hunger, and glitch transitions for the moments the report itself breaks down. This repo has all five layers. Everything below is ranked and role-tagged; scores are the 0–10 psychedelic-intensity scale from the full breakdown.
